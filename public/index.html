Welcome to Mobile Dev Onboarding

Your journey starts here. Below is everything you need to get started building with Flutter and Firebase.

1. Environment Setup

  • Install Flutter SDK and add it to your system PATH
  • Install Dart SDK (if separate)
  • Set up Android Studio or VS Code
  • Clone the project repo and run flutter pub get
  • Verify with a test run on emulator/device

2. Firebase Setup

  • Access the Firebase Console
  • Download google-services.json and/or GoogleService-Info.plist
  • Install Firebase CLI: npm install -g firebase-tools
  • Run firebase login and firebase init
  • Enable Auth, Firestore, and Storage

3. Project Knowledge

  • Read the README and contribution guidelines
  • Understand folder structure and navigation
  • Learn state management used (Provider, Riverpod, BLoC)
  • Review how Firebase is integrated across features

4. Development Standards

  • Follow coding style and naming conventions
  • Use Git strategy (feature/bugfix branches)
  • Conventional commit messages
  • Understand PR and review process
  • Write tests per Flutter testing guide
  • Use Flutter lints

5. Tools & Accounts

  • Get access to GitHub/GitLab, Firebase, Slack/Discord, Jira/Trello
  • Access CI/CD dashboards (Codemagic, GitHub Actions)
  • Run flutter format . for code styling

6. Initial Tasks

  • Fix a small issue or implement a feature
  • Push code and create a PR
  • Pass all tests and deploy to staging

7. Optional (But Recommended)

  • Document a module or process
  • Help improve the README or setup guide
  • Join team syncs or watch recordings
