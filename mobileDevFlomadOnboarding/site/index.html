Mobile Developer Onboarding

1. Environment Setup

  • Install Flutter SDK and add it to your system PATH
  • Install Dart SDK if needed separately
  • Set up Android Studio or VS Code
  • Clone the repo and run flutter pub get

2. Firebase Setup

  • Firebase Console
  • Set up Firebase CLI: firebase-tools
  • Initialize services: Auth, Firestore, Storage

3. Project Knowledge

  • Read the README
  • Understand the folder structure and modules
  • Learn how navigation and auth flows work
  • State management: Provider, Riverpod, BLoC etc.

4. Development Standards

  • Follow code style and naming conventions
  • Use conventional commits and our PR process
  • Write tests: Flutter testing guide
  • Install and follow Flutter lints

5. Tools & Accounts

  • Join repo (GitHub/GitLab)
  • Get access to Firebase, Slack/Discord, Jira/Trello
  • CI/CD tools like Codemagic or GitHub Actions

6. Initial Tasks

  • Fix a small bug or add a feature
  • Push changes and open a PR
  • Pass all tests and deploy to staging

7. Recommended

  • Document a module
  • Improve the README or add resources
  • Join team syncs or view past recordings

Welcome aboard and happy coding!
